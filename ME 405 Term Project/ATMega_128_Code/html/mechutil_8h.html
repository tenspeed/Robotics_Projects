Detailed Description

This file contains a set of enhancements to the GCC-AVR system which support some of the more helpful C++ features that aren't present or working well in the normal GCC-AVR distribution (though I have no idea why). The information came from an AVR-Freaks posting at:

  • http://www.avrfreaks.net/index.php?name=PNphpBB2&file=viewtopic&p=410870

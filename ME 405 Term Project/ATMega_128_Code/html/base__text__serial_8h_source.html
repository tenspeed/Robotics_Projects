147 /** This is a base class for lots of serial devices which send text over some type of
148  * communication interface. Descendents of this class will be able to send text over
149  * serial lines, radio modems or modules, and whatever other communication devices
150  * might be appropriate.
151  *
152  * The most important methods of this class are the overloaded "<<" operators which
153  * convert different types of numbers into printable strings, then print them using
154  * methods which are overloaded by descendent classes. Methods are provided which
155  * transmit most types of integers and floating point numbers; floating point numbers
156  * are only handled if <math.h> is defined near the beginning of base_text_serial.h
157  * because one often wants to turn off floating point support in order to save
158  * memory. Other classes may implement methods to print themselves to any descendent
159  * of this class; see stl_timer.h/cc for examples in which time stamps print the time
160  * conveniently to any serial device.
161  *
162  * The ability to transmit characters is provided by descendent classes. Methods
163  * which will be inherited or overridden by descendents include the following:
164  * \li ready_to_send () - Checks if the port is ready to transmit a character
165  * \li putchar() - Sends a single character over the communications line
166  * \li puts() - Sends a character string
167  * Other methods may optionally be overridden; for example clear_screen() is only
168  * needed by devices which have a screen, and some devices may be read-only or
169  * write-only, and some methods may not be needed.
170  */

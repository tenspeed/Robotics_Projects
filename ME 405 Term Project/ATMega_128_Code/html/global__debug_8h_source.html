2 /** \file global_debug.h
3  * This file contains a simple system to allow debugging printouts from anywhere in
4  * a simple AVR program. It sets up a global pointer to a base_text_serial object;
5  * this pointer will be set to point to any descendent of base_text_serial, such as
6  * an RS-232 style serial port (USB or classic), a radio modem, or even an SD card.
7  *
8  * The SERIAL_DEBUG macro should be defined in the Makefile so that global serial
9  * debugging is enabled. If SERIAL_DEBUG is not defined, the serial debugging port
10  * is not activated; this is good for production code as opposed to testing code.
11  *
12  * Revisions
13  * \li 01-31-2009 JRR Original file
14  * \li 02-08-2009 JRR Added code to enable this debugging under Linux
15  * \li 12-20-2009 JRR Added do_reboot() function
16  */

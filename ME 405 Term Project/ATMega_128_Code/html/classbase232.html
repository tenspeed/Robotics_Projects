Detailed Description

This class operates the hardware on an RS232 port in an AVR microcontroller. It sets up the port, checks for characters, and sends characters one at a time. Its descendent classes do useful things such as sending text messages and packets.

unsigned char  port_number = 0 
)

The constructor sets up the port with the given baud rate and port number.

This method sets up the AVR UART for communications. It enables the appropriate inputs and outputs and sets the baud rate divisor, and it saves pointers to the registers which are used to operate the serial port. Since some AVR processors have dual serial ports, this method allows one to specify a port number.

Parameters
baud_rateThe desired baud rate for serial communications. Default is 9600
port_numberThe number of the serial port, 0 or 1 (the second port numbered 1 only exists on some processors). The default is port 0

bool base232::is_sending ( void  )

This method returns true if the port is currently sending a character out.

This method checks if the serial port is currently sending a character. Even if the buffer is ready to accept a new character, the port might still be busy sending the last one; it would be a bad idea to put the processor to sleep before the character has been sent.

Returns
True if the port is currently sending a character, false if it's idle

1 /** task_read is the input section of the user interface for the polar plotter. It takes in the
2 * following commands:
3 * [h,H,?] print a help screen
4 * [o,O] home the machine
5 * [g,G] enable motor control
6 * [s,S] disable motor control (stop motors)
7 * [SPACE] Emergency stop
8 * [r,R] raise pen
9 * [l,L] lower pen
10 * [p,P] initiate K_p GAIN ENTRY MODE
11 * [i,I] initiate K_i GAIN ENTRY MODE
12 * [d,D] initiate K_d GAIN ENTRY MODE
13 * [c,C] initiate COORDINATE INPUT MODE to make a line
14 * [m,M] initiate COORDINATE INPUT MODE to make a point
15 * [q,Q] print current values of gains, encoders and set points
16 * [z,Z] reset PID controller (on both boards)
17 * [x,X] Make signiture
18 * COORDINATE INPUT MODES: Allows a user to enter values from 0 to 99.9 and accept one decimal place if desired
19 * GAIN ENTRY MODE: Allows a user to enter a number up to 5 digits. User is then promted on which motor to update.
20 * no enter key is required on this last step.
21 */

1 /** task_print is a state machine which handles the screen printout portion of the user
2 * interface for the polar plotter. State 0 checks to see if any prompts, errors or
3 * menus need to be printed. If so, the strings are copied to a buffer. State 1 prints the
4 * buffer one character at a time so as to be cooperative. When the string end is reached,
5 * an endline is printed and State is returned to 0. The exception to this rule is the case
6 * of the help menu. Because the menu is so long, it is divided into two strings. State two
7 * detects the end of the first half of the menu, copies the second half to the buffer and

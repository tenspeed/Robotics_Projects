70 /** da_motor.set_mode allows a user to choose a motor's rotational direction or apply
71 * unmodulated braking. These tasks are performed by passing motor_num a 1 or a 2 to
72 * select which motor's state to change and mode a 1, 2, or 3.

74 * @param mode: mode 1 = spin CW mode 2 = spin CCW mode 3 = unmodulated braking.

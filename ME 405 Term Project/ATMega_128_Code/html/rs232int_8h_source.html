4  * microcontroller. This version of the class uses the serial port receiver

6  * The port is used in "text mode"; that is, the information which is sent and
7  * received is expected to be plain ASCII text, and the set of overloaded left-shift
8  * operators "<<" in base_text_serial.* can be used to easily send all sorts of data
9  * to the serial port in a manner similar to iostreams (like "cout") in regular C++.
10  *

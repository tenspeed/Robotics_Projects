\subsection{Detailed Description}
This class operates the hardware on an R\-S232 port in an A\-V\-R microcontroller. It sets up the port, checks for characters, and sends characters one at a time. Its descendent classes do useful things such as sending text messages and packets.

The constructor sets up the port with the given baud rate and port number. 

This method sets up the A\-V\-R U\-A\-R\-T for communications. It enables the appropriate inputs and outputs and sets the baud rate divisor, and it saves pointers to the registers which are used to operate the serial port. Since some A\-V\-R processors have dual serial ports, this method allows one to specify a port number.

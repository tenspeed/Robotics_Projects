\subsection{Detailed Description}
This file contains a class which runs a task timer whose resolution is one microsecond. The timer is used to help schedule the execution of tasks' run() methods, and it is also used to keep track of real time in general. The run() methods can be run from a main loop by checking periodically if a task time has expired, or they can be called directly from a timer interrupt service routine.

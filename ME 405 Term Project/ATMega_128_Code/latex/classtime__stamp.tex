\subsection{Detailed Description}
This class holds a time stamp which is used to measure the passage of real time in the world around an A\-V\-R processor. This version of the time stamp implements a 32-\/bit time counter that runs at one megahertz. There's a 16-\/bit number which is copied directly from a 16-\/bit hardware counter and another 16-\/bit number which is incremented every time the hardware counter overflows; the combination of the two is a 32-\/bit time measurement. The time stamp is to be interpreted and printed as containing a whole number of seconds and a number of microseconds. The conversion between seconds and microseconds is determined by the C\-P\-U clock frequency, which is specified in the macro F\-\_\-\-C\-P\-U.
